# HoloGeo Project Page

Static project page for:

**HoloGeo: Mitigating Landmark Bias in Geo-localization via Evidence-Driven Reasoning**

## Local Preview

```bash
cd website
python3 -m http.server 4173
```

Open `http://localhost:4173/`.

## Deploy

This folder is GitHub Pages ready. Put the contents of `website/` on a `gh-pages` branch, or set GitHub Pages to serve this folder from the repository.

Before public release, replace:

- `https://is.gd/fZ0J30` with the final GitHub repository, dataset, or project links.
- The placeholder venue metadata with the camera-ready citation or DOI.
